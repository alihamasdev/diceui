## Anatomy

Import the parts, and compose them together.

```tsx
import * as TagsInput from "@diceui/tags-input";

<TagsInput.Root>
  <TagsInput.Label/>
  <TagsInput.Item >
    <TagsInput.Text />
    <TagsInput.Delete />
  </TagsInput.Item>
  <TagsInput.Input />
  <TagsInput.Clear />
</TagsInput.Root>
```
