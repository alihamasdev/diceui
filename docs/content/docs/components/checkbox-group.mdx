## Installation

### CLI

```package-install
npx shadcn@latest add "https://diceui.com/r/checkbox-group"
```

### Manual

<Steps>

## Anatomy

Import the parts, and compose them together.

```tsx
import * as CheckboxGroup from "@diceui/checkbox-group";

<CheckboxGroup.Root>
  <CheckboxGroup.Label />
  <CheckboxGroup.List>
    <CheckboxGroup.Item>
      <CheckboxGroup.Indicator />
    </CheckboxGroup.Item>
  </CheckboxGroup.List>
  <CheckboxGroup.Description>
</CheckboxGroup.Root>
```

## Features

- Support for controlled and uncontrolled states
- Custom validation with error handling
- Keyboard navigation
- Accessible by default with ARIA attributes
- Support for horizontal and vertical orientations
- Form integration with name support
- Disabled state support
- Required field support
- RTL support
- Orientation support
