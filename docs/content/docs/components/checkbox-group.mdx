## Features

- Support for controlled and uncontrolled states
- Custom validation with error handling
- Keyboard navigation
- Accessible by default with ARIA attributes
- Support for horizontal and vertical orientations
- Form integration with name support
- Disabled state support
- Required field support
- RTL support
- Orientation support
