# Frame

The Frame component helps create responsive containers that maintain specific aspect ratios, perfect for iframes, embedded content, videos, and other media content.

## Features

- Maintains aspect ratio across screen sizes
- Supports responsive scaling
- Handles overflow content gracefully
- Perfect for embedding iframes (YouTube, Vimeo, etc.)
- Zero dependencies

## Installation

```bash
npm install @diceui/frame
```

## Usage

### With iframes (YouTube, Vimeo, etc.)

```tsx
import { Frame } from "@diceui/frame"

export default function YouTubeEmbed() {
  return (
    <Frame ratio={16/9}>
      <iframe 
        src="https://www.youtube.com/embed/dQw4w9WgXcQ"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
        allowFullScreen
      />
    </Frame>
  )
}
```

### With Video Elements

```tsx
import { Frame } from "@diceui/frame"

export default function VideoPlayer() {
  return (
    <Frame ratio={16/9}>
      <video src="video.mp4" />
    </Frame>
  )
}
```

## API Reference

### Frame

The main component that creates an aspect-ratio maintained container.

#### Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `ratio` | `number` | `1` | The aspect ratio to maintain (width/height) |
| `children` | `ReactNode` | - | The content to display in the frame (iframe, video, etc.) |
| `fit` | `"contain" \| "cover"` | `"cover"` | How the content should fit |
| `position` | `string` | `"center"` | Position of content within frame |

### useFrame

A hook that provides frame calculations and measurements.

```tsx
import { useFrame } from "@diceui/frame"

function Component() {
  const { width, height, ratio } = useFrame(16/9)
  
  return (
    <div style={{ width, height }}>
      <iframe 
        src="https://www.example.com/embed"
        width={width}
        height={height}
        frameBorder="0"
        allowFullScreen
      />
    </div>
  )
}
```

## Common Aspect Ratios

- 16:9 (1.77778) - Standard widescreen video
- 4:3 (1.33333) - Traditional video
- 1:1 (1) - Square content
- 21:9 (2.33333) - Ultra-wide content
